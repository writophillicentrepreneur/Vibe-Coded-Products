# Vibe Coded Products

## Repository Overview
This repository serves as the central deployment hub for digital products, operational frameworks, and micro-applications architected through advanced natural language processing. The systems contained here bypass traditional development cycles, utilizing AI models to translate clinical logic and business parameters directly into functional code and digital assets.

## Core Directives
All products within this repository adhere to strict architectural standards:
* **High Signal-to-Noise:** Elimination of redundant code and generic interface elements.
* **Operational Utility:** Every application or framework solves a specific, measurable friction point in business operations, finance, or productivity.
* **Deterministic Output:** System prompts and architectures are engineered to produce reliable, clinical-grade results without hallucination or subjective fluff.

## Current Deployments

### 1. AI-Powered Educational eBook Generator
* **Architecture:** AWS Bedrock (Claude Opus 4.7) / Python / Next.js
* **Function:** An automated pipeline for generating structured, high-fidelity digital products. Utilizes frontier models with strict prompt caching to maintain clinical precision and deterministic formatting at scale.
* **Documentation:** `/docs/ebook-pipeline-architecture.md`


## Deployment Protocols
To deploy or test any of the interactive architectures contained in this repository:
1. Clone the repository to your local environment.
2. Navigate to the specific product directory.
3. Review the isolated `DEPLOYMENT.md` file within that directory for precise environmental variables, API requirements, and execution commands.

## Version Control & Maintenance
This repository is continuously updated as frontier AI models evolve. Legacy code that relies on deprecated models or outdated system prompts will be archived sequentially.
